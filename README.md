# 📝 Task Manager (Python)

This is a simple task manager developed in Python using basic programming concepts.

## 🚀 Features
- Add new tasks
- Remove tasks by number
- List all tasks
- Mark tasks as completed (✅)
- Input validation for user errors

## 🧠 Concepts Used
- Lists
- Functions
- Loops (`while`, `for`)
- Conditionals (`if/elif/else`)
- Exception handling (`try/except`)
- `enumerate()` for indexing
